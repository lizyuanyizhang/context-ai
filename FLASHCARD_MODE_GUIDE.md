# 🎴 闪卡学习模式使用指南

## ✨ 功能概述

闪卡学习模式是一个强大的单词记忆工具，通过翻转卡片的方式帮助用户高效学习生词本中的单词。

## 🎯 核心功能

### 1. **3D翻转效果**
- 点击卡片或按空格键翻转
- 正面显示单词原文和音标
- 背面显示翻译、语法和语境

### 2. **三种学习状态**
- **不认识 (1键)**：标记为"学习中"，会增加复习频率
- **认识 (2键)**：标记为"已掌握"，设置较长的复习间隔
- **已掌握 (3键)**：直接标记为已掌握，跳过当前学习

### 3. **智能复习算法**
- 根据回答正确率自动调整复习间隔
- 使用间隔重复算法：1天、3天、7天、14天、30天
- 优先显示需要复习的单词

### 4. **学习进度追踪**
- 实时显示学习进度百分比
- 显示当前单词位置（如：5 / 20）
- 记录学习次数和正确次数

### 5. **键盘快捷键**
- `Space`：翻转卡片
- `←` / `→`：上一个/下一个单词
- `1`：不认识
- `2`：认识
- `3`：已掌握
- `Esc`：退出学习模式

## 🚀 如何使用

### 进入学习模式

1. 打开生词本面板
2. 点击工具栏中的 **"学习模式"** 按钮（绿色按钮，带播放图标）
3. 开始学习！

### 学习流程

1. **查看单词原文**
   - 卡片正面显示单词原文和音标
   - 点击发音按钮可以听发音
   - 思考这个单词的意思

2. **翻转查看翻译**
   - 点击卡片或按空格键翻转
   - 查看翻译、语法和语境信息
   - 判断自己是否认识这个单词

3. **选择学习状态**
   - 点击按钮或按数字键选择状态
   - 系统会自动记录并调整复习时间
   - 自动跳转到下一个单词

4. **完成学习**
   - 完成所有单词后，可以选择重新开始或退出
   - 学习进度会实时更新

## 📊 学习状态说明

### 新单词 (new)
- 默认状态
- 首次学习时会显示

### 学习中 (learning)
- 选择"不认识"后标记
- 会增加复习频率
- 下次复习时间较短（1-3天）

### 已掌握 (mastered)
- 选择"认识"或"已掌握"后标记
- 复习间隔较长（7-30天）
- 正确率高的单词会设置更长的间隔

## 🎨 UI特性

- **Apple + Forest风格设计**：清新自然的界面
- **流畅的3D翻转动画**：视觉体验优秀
- **实时进度显示**：学习进度一目了然
- **响应式布局**：适配不同屏幕尺寸

## 💡 学习建议

1. **每天坚持学习**：建议每天学习10-20个单词
2. **诚实回答**：不要欺骗自己，诚实选择"认识"或"不认识"
3. **利用发音**：多听发音有助于记忆
4. **查看语法和语境**：理解用法比单纯记忆更重要
5. **定期复习**：系统会自动提醒需要复习的单词

## 🔧 技术实现

### 数据结构

每个单词现在包含以下学习相关字段：

```typescript
{
  masteryStatus?: 'new' | 'learning' | 'mastered'  // 掌握状态
  studyCount?: number                                // 学习次数
  correctCount?: number                              // 正确次数
  nextReviewAt?: number                              // 下次复习时间（时间戳）
  lastStudiedAt?: number                             // 最后学习时间（时间戳）
}
```

### 复习算法

- 根据正确率选择复习间隔
- 正确率 >= 80%：较长间隔（14-30天）
- 正确率 >= 60%：中等间隔（7天）
- 正确率 >= 40%：较短间隔（3天）
- 正确率 < 40%：最短间隔（1天）

## 🐛 常见问题

### Q: 为什么有些单词不显示在学习模式中？
A: 学习模式优先显示需要复习的单词。如果单词已经掌握且未到复习时间，不会显示。

### Q: 可以重置学习状态吗？
A: 目前不支持批量重置，但可以删除单词后重新添加。

### Q: 学习数据会丢失吗？
A: 不会。所有学习数据都保存在 `chrome.storage.local` 中，即使关闭浏览器也不会丢失。

### Q: 可以自定义复习间隔吗？
A: 目前使用固定的间隔算法，未来可能会添加自定义选项。
