# 📸 快速截图指南

## 🚀 快速开始（5分钟搞定）

### 步骤1：打开演示页面

```bash
# 在浏览器中打开
open screenshot-demo.html

# 或直接双击文件
```

### 步骤2：调整浏览器窗口

1. **打开开发者工具**：按 `F12` 或 `Cmd+Option+I`（Mac）
2. **点击设备工具栏**：按 `Ctrl+Shift+M`（Windows）或 `Cmd+Shift+M`（Mac）
3. **设置尺寸**：
   - 宽度：1280px
   - 高度：800px
   - 缩放：100%

### 步骤3：截图1 - 主要功能

1. **选中英语文本**：
   ```
   The quick brown fox jumps over the lazy dog.
   ```
2. **等待浮动按钮出现**
3. **点击"翻译"按钮**
4. **等待翻译完成**
5. **截图**：
   - Chrome：`Ctrl+Shift+P` → "Capture full size screenshot"
   - 或使用系统截图工具

### 步骤4：截图2 - 生词本

1. **先保存几个单词**（在截图1中点击"保存到生词本"）
2. **点击右下角的生词本按钮**
3. **截图生词本界面**

### 步骤5：截图3 - 闪卡学习

1. **在生词本中点击"闪卡学习"**
2. **等待闪卡加载**
3. **截图闪卡界面**

### 步骤6：优化图片

1. **使用 TinyPNG 压缩**：https://tinypng.com/
2. **确保文件大小 < 1MB**
3. **检查清晰度**

## 📋 截图检查清单

- [ ] 截图2：生词本功能

## 🎯 推荐截图工具

### Chrome内置工具（推荐）

1. 按 `F12` 打开开发者工具
2. 按 `Ctrl+Shift+P`（Mac: `Cmd+Shift+P`）
3. 输入 "screenshot"
4. 选择 "Capture full size screenshot"

### 浏览器扩展

- **Awesome Screenshot**：https://chrome.google.com/webstore/detail/awesome-screenshot
- **Nimbus Screenshot**：https://chrome.google.com/webstore/detail/nimbus-screenshot

### 在线工具

- **TinyPNG**：压缩图片 https://tinypng.com/
- **Screely**：添加浏览器框架 https://www.screely.com/
