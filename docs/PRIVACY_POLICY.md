# Context AI 隐私政策

**最后更新：2025-02**

## 一、概述

Context AI（以下简称「本扩展」）是一款外语学习浏览器扩展，致力于保护用户隐私。本政策说明我们如何收集、使用和存储数据。

## 二、数据存储

### 2.1 本地存储

以下数据**仅存储在您设备本地**，不会上传到任何第三方服务器：

- **生词本数据**：您保存的单词、翻译、语法等学习内容
- **API 配置**：您填写的服务商 API Key 和设置（如通义千问、DeepSeek 等）
- **学习进度**：闪卡复习状态、FSRS 算法相关数据

存储方式：浏览器扩展的 `chrome.storage.local` / `browser.storage.local`。

### 2.2 我们不收集

- 不收集个人身份信息（姓名、邮箱、电话等）
- 不收集浏览历史或网页内容
- 不进行行为追踪或数据分析

## 三、数据外发

### 3.1 翻译相关

当您使用翻译功能时：

- **预翻译（LibreTranslate）**：您选中的文字会发送到 LibreTranslate 的公开接口，以获得快速初译
- **大模型增强**：若您启用了大模型，选中的文字会发送到**您自行配置**的 API 服务商（如通义千问、DeepSeek、OpenAI 等）

上述数据由您选择的服务商按其隐私政策处理。本扩展不控制、不存储这些第三方传输的内容。

### 3.2 语音朗读

语音朗读由浏览器内置或系统 TTS 完成，无需上传数据。

## 四、权限说明

| 权限 | 用途 |
|------|------|
| `storage` | 本地保存生词本、API 配置、学习进度 |
| `activeTab` | 获取当前页选中文字以翻译 |
| `scripting` | 在页面注入翻译面板等 UI |
| `<all_urls>` | 在任何网页上使用选中翻译功能 |

## 五、第三方服务

- **LibreTranslate**：免费机器翻译服务，详见其隐私政策
- **您配置的 API 服务商**：通义千问、DeepSeek、豆包、Kimi、智谱、Gemini、OpenAI、Grok 等，各有独立隐私政策

## 六、联系与更新

- 本政策可能随功能更新而调整，重大变更会在扩展内或发布说明中告知
- 如有隐私相关问题，请通过扩展的 GitHub 仓库 Issue 反馈
