# 📸 Chrome Web Store 商店截图准备指南

## 📋 Chrome Web Store 截图要求

### 尺寸要求

- **最小尺寸**：1280 x 800 像素
- **推荐尺寸**：1280 x 800 或 1920 x 1200
- **格式**：PNG 或 JPEG
- **文件大小**：建议 < 1MB（压缩后）

### 数量要求

- **至少需要**：1 张截图
- **推荐数量**：3-5 张截图
- **最多**：5 张截图

## 🛠️ 截图方法

### 方法1：使用浏览器开发者工具（推荐）

#### 步骤1：准备演示页面

1. **创建一个测试HTML页面**（见下方 `screenshot-demo.html`）
2. **加载插件到Chrome**
3. **打开测试页面**

#### 步骤2：触发插件功能

1. **选中文本**，显示浮动按钮
2. **点击翻译**，显示翻译面板
3. **调整浏览器窗口**到合适大小

#### 步骤3：截图

**Chrome浏览器**：
1. 按 `F12` 打开开发者工具
2. 按 `Ctrl+Shift+P`（Mac: `Cmd+Shift+P`）
3. 输入 "Capture screenshot"
4. 选择 "Capture full size screenshot"（完整页面截图）

**或使用扩展工具**：
- **Awesome Screenshot**：https://chrome.google.com/webstore/detail/awesome-screenshot
- **Nimbus Screenshot**：https://chrome.google.com/webstore/detail/nimbus-screenshot

### 方法2：使用系统截图工具

#### macOS

1. **调整浏览器窗口**到 1280 x 800
2. **按 `Cmd+Shift+4`** 选择区域截图
3. **或按 `Cmd+Shift+3`** 全屏截图

#### Windows

1. **调整浏览器窗口**到 1280 x 800
2. **按 `Win+Shift+S`** 使用截图工具
3. **或使用 Snipping Tool**

### 方法3：使用在线工具

- **Screely**：https://www.screely.com/（添加浏览器框架）
- **Mockup World**：https://www.mockupworld.co/（浏览器模板）

## 🎨 截图优化技巧

### 1. 选择合适的文本

**好的示例**：
- ✅ 完整的句子（不是单个单词）
- ✅ 有语法结构的句子
- ✅ 常见的外语学习场景

**避免**：
- ❌ 太长的文本（超过500字符）
- ❌ 敏感内容
- ❌ 无意义的文本

### 2. 调整浏览器窗口

**推荐设置**：
- **宽度**：1280px
- **高度**：800px
- **缩放**：100%（不要缩放）

**如何设置**：
1. 打开开发者工具（F12）
2. 点击设备工具栏图标（或按 `Ctrl+Shift+M`）
3. 选择 "Responsive"
4. 设置宽度：1280px，高度：800px

### 3. 美化截图

**使用工具**：
- **Figma**：添加标注和箭头
- **Canva**：添加文字说明
- **Photoshop**：专业编辑

**添加元素**（可选）：
- 箭头指向重要功能
- 文字说明关键特性
- 高亮显示核心功能

### 4. 压缩图片

**工具**：
- **TinyPNG**：https://tinypng.com/
- **Squoosh**：https://squoosh.app/

**目标**：
- 文件大小 < 1MB
- 保持清晰度
