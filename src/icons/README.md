# 图标文件说明

请在此目录下放置以下尺寸的图标文件：

- `icon16.png` - 16x16 像素（用于浏览器工具栏）
- `icon48.png` - 48x48 像素（用于扩展管理页面）
- `icon128.png` - 128x128 像素（用于 Chrome 网上应用店）

## 临时解决方案

如果暂时没有图标，可以：

1. 使用在线图标生成器（如 [Favicon Generator](https://www.favicon-generator.org/)）
2. 创建简单的占位图标
3. 或者暂时注释掉 manifest.json 中的 icons 字段

## 图标设计建议

- 使用简洁的设计，在小尺寸下也能清晰识别
- 使用与"翻译"、"AI"、"学习"相关的图标元素
- 建议使用蓝色或绿色等友好的颜色
